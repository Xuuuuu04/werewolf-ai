# 🎨 游戏界面优化说明

## ✨ 主要改进

### 1. 彩色控制台输出

使用 `colorama` 库实现彩色控制台，提升游戏体验：

#### 角色颜色主题
- 🐺 **狼人** - 红色 (强调危险)
- 🔮 **预言家** - 青色 (智慧与洞察)
- 🧪 **女巫** - 洋红色 (神秘与力量)
- 🛡️ **守卫** - 蓝色 (保护与防御)
- 🔫 **猎人** - 绿色 (行动与正义)
- 👤 **村民** - 白色 (中立与普通)

#### 阶段颜色区分
- 🌙 **夜晚** - 蓝色背景
- ☀️ **白天** - 黄色高亮
- 🗳️ **投票** - 青色提示
- 💬 **发言** - 绿色显示

#### 状态提示
- ✅ **成功** - 绿色
- ⚠️ **警告** - 黄色
- ❌ **错误** - 红色
- ℹ️ **信息** - 青色

### 4. 狼人杀专业术语集成

#### 核心术语
- **金水** - 预言家查验为好人的玩家
- **查杀** - 预言家查验为狼人的玩家
- **银水** - 被女巫解药救活的玩家
- **悍跳** - 狼人冒充预言家等神职
- **倒钩** - 狼人针对同伙制造对立
- **站边** - 选择相信哪一方预言家
- **归票** - 统一投票方向
- **抿身份** - 推测玩家真实身份
- **自刀** - 狼人杀害自己队友
- **推人** - 引导投票某玩家

#### AI提示词增强
在游戏说明中加入专业术语，帮助AI理解和使用：

```
【狼人杀专业术语】
- 金水：预言家查验为好人的玩家，是真实身份的证明
- 查杀：预言家查验为狼人的玩家，需要推出局外
- 银水：被女巫解药救活的玩家，可信度较高
- 悍跳：狼人冒充预言家等神职，是狼人的重要战术
...

【游戏策略提示】
- 预言家应该适时报出"金水"和"查杀"信息
- 女巫需要判断是否有狼人"自刀"的可能性
- 村民要注意识别"悍跳"，通过逻辑"站边"真预言家
- 狼人可以使用"悍跳"、"倒钩"等战术迷惑好人
- 投票时要注意"归票"，避免平票导致PK
```

## 🎮 使用示例

### 启动游戏
```bash
python start_game.py
```

### 人类玩家操作

1. **发言阶段**
   - 输入发言内容
   - 可以使用专业术语（如："我是预言家，1号是我的金水"）

2. **动作阶段**
   - 输入数字索引选择动作（推荐）
   - 或输入完整动作字符串

3. **查看提示**
   - 每个阶段都有相应的策略提示
   - 注意观察彩色高亮的重要信息

## 🔧 技术实现

### 核心类：ConsoleUI

位置：`werewolf/helper/console_ui.py`

主要方法：
- `print_header()` - 打印标题头部
- `print_player_info()` - 打印玩家信息卡片
- `print_action_list()` - 打印彩色动作列表
- `print_game_log()` - 自动高亮游戏日志
- `print_tips()` - 打印阶段提示
- `print_success/error/warning/info()` - 状态提示

### 集成点

1. **human_agent.py** - 人类玩家交互界面
2. **run_battle.py** - 游戏开始/结束提示
3. **prompt_template_v0.py** - AI提示词增强
